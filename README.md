# Web Chat App on Js

### [View the demo](https://web-chat-app-js.herokuapp.com/)

## List of included features

+ Online user 
+ See who is typing...
+ FrontEnd script on Modev-View-Controller(MVC) pattern

## Installation
```bash
git clone https://github.com/LavrL/webChatApp-JS.git
cd webChatApp-JS
npm i
```

## Running 
```bash
npm run start
```

## Used:
+ Socket.io
+ Jest
+ MongoDB
